# IO

## IO流的分类

按操作单位不同：

+ 字节流（8 Bit ）010101...型:处理图片、声音，视频
+ 字符流 （16 Bit ）char型 ： 处理文本数据（因为里面是一个一个的字符）

按流的角色不同：

+ 节点流：
+ 处理流：

按数据的流向：

+ 输入流
+ 输出流

## 抽象基类

| 抽象基类 |    字节流    | 字符流 |
| :------: | :----------: | :----: |
|   输入   | InputStream  | Reader |
|   输出   | OutputStream | Writer |

java中IO流共涉及40多个子类，但是都是从4个**抽象**基类派生的  （抽象类不能实例化，抽象类不一定有抽象方法）
